I'm gonna tell you some basic rules that will help you to make attractive sketches.

    BASICS
  • Use 2b pencil while sketching and 6b while blending
  • Make sure that you do not draw so dark that it gets very hard to rub.
  • Do shading with blenders rather than using direct pencil (increases realism in your sketch)
  • Stay calm while sketching
    •

These were some basic key features that you must keep in mind everytime while painting.
